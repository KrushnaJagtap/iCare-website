About This Website !


This is a health-check website created to reduce the stress and pressure in people's life. The main aim of this website is to provide all the necessary information related to health and diet in one place so that people have don't have to waste their time and money.

Proper guidance and true information is provided to the users of this website. The main features of this webiste are: COVID Vaccination availability check, Nutrition and diet plan for several illness/diseases and a map has been embeded to guide the patients to the nearby hospital.

These features will not only save the time but also reduce crowd in the public place which is our top priority these days.
